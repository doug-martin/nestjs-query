# `@nestjs-query/core`

Core package that contains interfaces for types used by [`@nestjs-query/query-graphql`](../query-graphql) and [`@nestjs-query/query-typeorm`](../query-typeorm)

## Installation

```sh
npm i @nestjs-query/core
```
